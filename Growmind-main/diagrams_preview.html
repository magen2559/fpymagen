5. Activity Diagram (Focus Session)

Shows the specific activities during a focus timer session.
stateDiagram-v2
    [*] --> Idle
    Idle --> ConfiguringSession : Set timer
    ConfiguringSession --> SessionActive : Start
    
    state SessionActive {
        [*] --> CountingDown
        CountingDown --> Paused : User pauses
        Paused --> CountingDown : User resumes
        CountingDown --> Abandoned : User aborts
        CountingDown --> Completed : Timer reaches 0
    }
    
    SessionActive --> CalculatingRewards : Completed
    SessionActive --> RecordingFailure : Abandoned
    
    CalculatingRewards --> UpdatingProfile : Add XP
    RecordingFailure --> ApplyingPenalties : Deduct Health
    
    UpdatingProfile --> [*]
    ApplyingPenalties --> [*]

6. Sequence Diagram

Demonstrates the data exchange between components when completing a focus session.
sequenceDiagram
    actor User
    participant App as React Native App
    participant Timer as Timer logic
    participant DB as Supabase backend
    
    User->>App: Configure & Start Focus Session
    App->>DB: Create Session Record (status='active')
    App->>Timer: Start Countdown
    Timer-->>App: Tick over time
    
    alt Session Finished
        Timer->>App: Reach 0
        App->>DB: Update Session (status='completed')
        App->>DB: Add XP to Profile
        App->>DB: Update Plant properties
        App-->>User: Render Success UI / Level Up
    else User Aborts
        User->>App: Press "Give Up"
        App->>Timer: Stop Countdown
        App->>DB: Update Session (status='abandoned')
        App-->>User: Render Failure UI
    end
